# Site - Bibliadeep
Olá, esse repo é uma ramificação do meu outro projeto: "Projeto-HTML"
Mas dessa vez com um tema
## Objetivo
Eu, Natan, como cristão e em formação para me tornar um TI e Dev, tive a ideia de produzir uma pagina dedicada a armezenas estudo biblicos.
Essa é uma forma de juntar meus estudos com a vida espiritual, alem de ter uma boa utilidade de armazenar tudo o que Deus tem falado atraves da sua Palavra.
### Nome
<strong>Bibliadeep</strong> é o nome que eu pensei pra esse projeto. Deep é uma palavra em inglês que significa profundo, juntando com bĺblia o nome remete a profundidade das escrituras.
